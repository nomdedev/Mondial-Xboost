# Glosario de Dominio

| Término | Definición |
|---------|------------|
| Predictor | Componente que produce distribución de probabilidad sobre resultados |
| Fixture | Partido programado |
| MatchContext | Datos agregados pre-partido |
| Availability | Estado de disponibilidad de jugador |
| Odds | Probabilidad implícita del mercado |
| Ensemble | Combinación ponderada de predictores |
| Simulation | Monte Carlo del torneo |
| Brier Score | Error cuadrático medio de pronósticos |
| Log-Loss | Penalización de predicciones confiadas incorrectas |
| RPS | Ranked Probability Score |
| ECE | Expected Calibration Error |
| Model Manifest | Registro reproducible de un modelo entrenado: dataset hash, features, hiperparámetros, métricas, hashes de artefactos |
| Canonical | Componente oficial y activo del pipeline; opuesto a experimental/no integrado |
